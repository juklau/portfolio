Mes applications et mes réalisations professionnelles

Qu'est-ce qu'une réalisation professionnelle ?

Dans le cadre de mon parcours, les réalisations professionnelles (RP) ont été des expériences enrichissantes qui m'ont permis de relier l'apprentissage théorique à la pratique professionnelle. À travers les différents thèmes abordés, j'ai développé des compétences techniques et comportementales essentielles, en affinant mon sens de la collaboration et de la communication interpersonnelle. Ces projets m'ont aidée à mieux comprendre les exigences du secteur, tout en renforçant mon aptitude à fournir des solutions adaptées aux besoins professionnels.

Applications professionnelles effectuées en première année
Application de Chat en Temps Réel

Communication Instantanée et Collaborative

Source: http://imaginland.centerblog.net/3124719-Salon-de-t-chat-amities

C'est une application de chat qui permet aux utilisateurs de discuter simultanément, en temps réel, avec plusieurs personnes. Grâce à une interface intuitive, les messages sont envoyés au serveur, qui les redistribue instantanément à tous les participants pour assurer une conversation fluide et synchronisée.

Mini CRM avec Airtable

Gestion de contact

Applications et réalisations professionnelles effectuées en deuxième année
MediaStock

Application de gestion de matériel

C'est une application web de gestion de prêts de matériel multimédia pour un établissement scolaire. Elle permet à l'équipe administrative de suivre et enregistrer les emprunts grâce à une interface mobile-first et un système de QR codes, remplaçant efficacement les fiches papier.

CryptoVault

Coffre-fort numérique sécurisé

Application de coffre-fort numérique permettant de déposer, organiser et partager des fichiers de manière sécurisée. Back-end PHP (Slim + Medoo) + BDD, authentification JWT, gestion des dossiers/fichiers, quotas, liens de partage et journalisation des téléchargements, avec un client JavaFX (gestion/upload) et un client Web (téléchargement via lien).

Météo 7 jours

Prévisions pour les villes de France

Météo 7 jours est une application web permettant de consulter les prévisions météorologiques via une API, avec une interface interactive intégrant graphiques, statistiques et carte pour une visualisation claire des données.

Outils/ Logiciels utilisés

Les divers outils et logiciels que j'ai découverts et utilisés au cours de ma formation.

  • Eclipse

    Java Eclipse est un environnement de développement intégré ouvert pour le langage de programmation Java.

  • Figma

    Figma est un outil qui aide les équipes de conception et de développement à concevoir et affiner en collaboration des produits exceptionnels dans un seul fichier de conception partagé.

  • Filius

    Filius est un logiciel de simulation de réseaux informatiques qui permet de créer son propre réseau, de le configurer, de simuler et de visualiser les échanges d'informations.

  • Git Bash

    Git Bash est une application pour les environnements Microsoft Windows qui fournit une couche d'émulation pour une expérience en ligne de commande Git.

  • GitHub

    GitHub est une entreprise de développement et services logiciels. Il facilite la collaboration à une échelle mondiale sur les projet.

  • MAMP

    Un MAMP (macOS, Apache, MySQL et PHP/Python) est une pile de logiciels qui permet l'utilisation d'un serveur Web local.

  • MySQL

    MySQL est un système de gestion de bases de données relationnelles.

  • Thonny

    Thonny est un logiciel libre et gratuit permettant l'écriture et l'exécution de programmes écrits en Python.

  • Visual Studio Code

    Visual Studio Code (VSCode) est un éditeur de code source et un environnement de développement intégré (IDE) de Microsoft.

  • Laragon

    Laragon est un environnement de développement local tout-en-un pour créer et tester des applications web (PHP, MySQL, Apache/Nginx).

  • VirtualBox

    VirtualBox est un logiciel de virtualisation permettant d'exécuter des systèmes d'exploitation virtuels sur un ordinateur.

  • IntelliJ IDEA

    IntelliJ IDEA est un environnement de développement intégré (IDE) utilisé principalement pour le développement Java.

  • Docker Desktop

    Docker Desktop est une application qui facilite l'utilisation de Docker pour la conteneurisation grâce à une interface graphique intégrée.
